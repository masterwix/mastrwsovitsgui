    Icon number 1, the first on the left, represents the svc pre-resample command.

Ensure that all your WAV files are in mono format. In the configuration, make sure that your raw dataset location is correctly set. You'll only see the config.json file created in /configs/44k after running the second step. Before this, the Advanced Configuration won't be accessible.
